Instructions for the game

For this game, the objective is to steal the paintings of the Rijksmuseum and not to get caught. If you are caught it's game over. Hover over the painting to show the name of the painting, if you are willing to steal the painting. Write or select the name of the painting on the input to the left. Press steal if you want to steal the selected paintings.
